ビット演算子

　数値を2進数で表して、各ビットを演算します。

演算子 意味
and 各ビットをand演算します。両方のビットが1のときだけ、結果が1になります。
or 各ビットをor演算します。左辺、右辺どちらか一方でも1のときは、結果が1になります。
not 各ビットを逆転します。0は1に、1は0になります。
xor ビットごとの排他的論理和。片方が0で、もう片方が1のときだけ、結果が1になります

　パソコン内部では、数値はすべて2進数で表現されています。ビットとは、2進数にしたときの各桁を表します。例えば、10進数で100を2進数で表すと、1100100になります。
　次がビット演算の例です。

  var
    X, Y, Ans: Integer;
  begin
    X := 100;  { 2進数では、1100100 }
    Y := 78;   { 2進数では、1001110 }

    Ans := X and Y;    { Ans = 68  (1000100) }
    Ans := X or Y;     { Ans = 110 (1101110) }
    Ans := not X;      { Ans = -2147483547 (11111111111111111111111110011011) }
    Ans := X xor Y;    { Ans = 42  (0101010) }
  end;

　※not Xの桁が大きいのは、実際のInteger型は4バイト(32ビット)であり、そのすべてのビットが反転するため。
