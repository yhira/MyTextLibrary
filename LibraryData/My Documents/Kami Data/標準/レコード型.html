レコード型

　レコード型は、複数の変数をまとめて扱うための型です。例えば名前、年齢、血液型の個人情報を記憶することを考えてみます。個人情報がそれぞれバラバラになっているよりも、一つにまとめると便利です。
　そんなとき、レコード型を使います。レコード型は次のように宣言します。

  type
    型名 = record
      メンバ名: データ型;
      メンバ名: データ型;
　　　　　・
　　　　　・
      メンバ名: データ型;
    end;

　個人情報を記憶するデータ型 TParsonを宣言してみます。

  type
    TParson = record
      Name : string;
      Old  : Integer;
      Blood: (blA, blB, blO, blAB);
    end;

　Name(名前)、Old(年齢)、Blood(血液型)のデータが、TParson型にまとめられました。レコード型のメンバとは、レコード型に含まれる変数のことです。TParson型の変数は、ほかの型の変数と同様に宣言します。

  var
    Parson: TParson;

　それぞれのメンバにデータを代入するときは、．(ピリオド)を付けます。次の例を見てください。ピリオドの前後に空白を入れることはできません。

  Parson.Name := '山田';
  Parson.Old := 30;
  Parson.Blood := blA;
