論理演算子

　論理型で真偽の判定に使います。主にif文で使用します。

演算子 意味
and 左辺と右辺がどちらも真(True)の場合、結果が真(True)になります
or 左辺と右辺のどちらか一方でも真(True)の場合、結果が真(True)になります
not 論理値を逆転する
xor 排他的論理和。片方が真(True)で、もう片方が偽(False)ときだけ、真(True)になります

　論理演算子の演算結果は、次の例のようになります。

  var
    X, Y, Z: Boolean;
    Ans: Boolean;
  begin
    X := True;
    Y := True;
    Z := False;

    Ans := X and Y;    { Ans = True になる }
    Ans := X and Z;    { Ans = False になる }
    Ans := X or Z;     { Ans = True になる }
    Ans := not X;      { Ans = False になる }
    Ans := X xor Z;    { Ans = True になる }
  end;
