リッチエディットでスクロールさせる

リッチエディットでテキストをスクロールさせるには EM_LINESCROLL メッセージを使用します。

■リッチエディットをスクロールさせる例
  procedure TForm1.Button1Click(Sender: TObject);
  begin
    RichEdit1.SetFocus;
    if  CheckBox1.Checked then
    begin
      { 上にスクロール }
      RichEdit1.Perform(EM_LINESCROLL, 0, -1);
    end else begin
      { 下にスクロール }
      RichEdit1.Perform(EM_LINESCROLL, 0, 1);
    end;
  end;
