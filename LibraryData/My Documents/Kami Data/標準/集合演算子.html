集合演算子

　集合型の演算に使用します。in以外にも集合演算子はありますが、ここでは説明を省きます。

演算子 意味
in 集合に要素が含まれているかチェックします。含まれているときは真(True)、含まれていないときは偽(False)の結果になります。

　in演算子は、次の書式で使用します。

  要素 in 集合型

　使用例は次のとおりです。

  var
    Ans: Boolean;
    Fruit: set of (frApple, frOrange, frGrape);
  begin
    Fruit := [frApple, frGrape];

    Ans := frOrange in Fruit;  { FruitにfrOrangeが含まれていないので、Ans = False }
    Ans := frApple in Fruit;   { frAppleは含まれているので、Ans = True になる }
  end;
