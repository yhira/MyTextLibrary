
function MessageBox(const Text, Caption: PChar; Flags: Longint = MB_OK): Integer;

説明

MessageBox メソッドは，メッセージが 1 つとボタンが 1 つ以上ある汎用ダイアログボックスを表示します。Caption はオプションであり，ダイアログボックスのキャプションを指定します。
MessageBox メソッドは Windows API MessageBox 関数をカプセル化します。TApplication を使って MessageBox をカプセル化すると，Windows API 関数で必要となるウィンドウハンドルのパラメータが自動的に提供されます。
Text パラメータの値はメッセージです。255 文字を超えるメッセージも指定でき，長いメッセージはメッセージボックス内で自動的に折り返されます。

Caption パラメータの値はダイアログボックスのタイトルバーに表示されるキャプションです。キャプションは 255 字を超えることができますが折り返されません。キャプションを長くすると幅の広いメッセージボックスになります。

Flags パラメータは，メッセージボックスに表示するボタンとメッセージボックスの動作（戻り値）を指定します。次の表に，指定できる値を示します。必要な効果を得るため複数の値を指定できます。

値	意味

MB_ABORTRETRYIGNORE	メッセージボックスには，[中止]，[再試行]，および [無視] の 3 つのプッシュボタンがある
MB_OK	メッセージボックスには，プッシュボタンの [OK] がある。デフォルト値
MB_OKCANCEL	メッセージボックスには，[OK] と [キャンセル] の 2 つのプッシュボタンがある
MB_RETRYCANCEL	メッセージボックスには，[再試行] と [キャンセル] の 2 つのプッシュボタンがある
MB_YESNO	メッセージボックスには，[はい] と [いいえ] の 2 つのプッシュボタンがある
MB_YESNOCANCEL	メッセージボックスには，[はい]，[いいえ]，および [キャンセル] の 3 つのプッシュボタンがある

メッセージボックスの作成に必要なメモリがない場合，MessageBox メソッドは 0 を返します。それ以外の場合は，次の値のうちのいずれかを返します。

値	数値	意味

IDOK	1	[OK] ボタンを選択した
IDCANCEL	2	[キャンセル] ボタンを選択した
IDABORT	3	[中止] ボタンを選択した
IDRETRY	4	[再試行] ボタンを選択した
IDIGNORE	5	[無視] ボタンを選択した
IDYES	6	[はい] ボタンを選択した
IDNO	7	[いいえ] ボタンを選択した
