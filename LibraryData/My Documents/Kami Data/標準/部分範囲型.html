部分範囲型

　整数型などで特定の範囲の値を代入できるようにするための型です。例えば次の宣言があったとします。

  var
    Year: Word;

　YearはWord型なので、0から65535までの数値を代入できます。しかし、Yearには1900から2100までの数値に限定したいとします。そのときに、部分範囲型を使います。部分範囲型は次のように宣言します。

  var
    Year: 1900..2100;

　このようにすると、Yearには1900から2100までの整数だけを代入できます。部分範囲型も、ユーザ定義型にすることができます。ユーザ定義型は次のように宣言します。

  type
    TYear = 1900..2100;

  var
    Year: TYear;

　部分範囲型には、整数型、文字型、列挙型を使用できます。次の例を見てください。

  type
    TMyColor = (mcRed, mcBlue, mcGreen, mcYellow);
    TBubun = mcBlue..mcYellow;
    TAlpha = 'A'..'Z';

  var
    Color: TMyColor;
    Bubun: TBubun;
    Alpha: TAlpha;

　ColorはTMyColor型で、これにはcmRed、mcBlue、mcGreen、mcYellowの値を代入できます。TBubun型はその中で、mcBlueからmcYellowまでの値を使えます。つまり、mcBlue、mcGreen、mcYellowを代入できます。もし、BubunにmcRedを代入しようとしたら、エラーになります。
　Alphaは、アルファベット大文字のAからZまでに限定しています。それ以外の文字を代入しようとするとエラーになります。また、文字は上記の例のように、’(シングルクォーテーション)で囲む必要があります。
